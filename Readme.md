# Практическая работа: PIP, LOGGING, MIGRATION

### 1. Описание приложения
Приложение разработано для изучения основ Python и практики настройки рабочего окружения

### 2. Структура проекта
*   **app.py** — основной файл с логикой проекта 
*   **migrate.py** — скрипт для миграции базы данных (создание и обновление структуры таблиц).
*   **tasks.db** — файл базы данных SQLite, в котором хранятся все рабочие записи.
*   **report.txt** — текстовый файл, в который записываются итоги выполнения задач.
*   **app.log** — журнал событий, где фиксируются этапы работы и ошибки.
*   **Readme.md** — подробная инструкция к проекту.

### 3. Зависимости
Для корректного отображения данных необходимо установить библиотеку `tabulate`. 
Выполните команду в терминале:
pip install tabulate

### 4. Нужно ли выполнять миграцию – и если да, как именно
Да, выполнение миграции обязательно.
Это необходимо для создания структуры таблиц в базе данных tasks.db. 
Чтобы выполнить миграцию, запустите скрипт через терминал:bashpython migrate.py
Используйте код с осторожностью.

### 5. Как запустить приложение
Откройте файл app.py.
Нажмите клавишу F5 на клавиатуре или кнопку Run

### 6. Где посмотреть результат работы (лог, отчёт)
Результаты работы приложения сохраняются в основной директории проекта: файл report.txt app.log
